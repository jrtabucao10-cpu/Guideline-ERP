# Guideline ERP

Interior fit-out and joinery factory ERP for Guideline Interiors and Guideline Decor.

## Workflow covered

1. Guideline Interiors creates quotations.
2. Approved quotation becomes a project.
3. If joinery is required, Guideline Interiors issues a job order to Guideline Decor.
4. Guideline Decor creates purchase requests linked to the job order and project.
5. Purchases can be recorded with LPO or without LPO.
6. Labour hours are tracked per job order to monitor true production cost.
7. Reports show materials, labour, other cost, and margin per job order/project.

## First version

This version saves data in the browser using local storage. For multi-user online use, connect it later to Supabase with company permissions, approvals, and shared database.
